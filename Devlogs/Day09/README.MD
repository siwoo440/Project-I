## 오늘의 목표
"탐색 → 보물 획득 → 마차 적재 → 탈출"이라는 게임의 핵심 루프를 처음으로 한 바퀴 완성한다.

## 한 일

### 1. InventorySystem 확장 — 꺼내기 API
- `TakeSelected()`: 현재 손에 든(두손 우선, 없으면 선택 슬롯) 것을 인벤토리에서 빼내 반환 (마차 적재용)
- `TakeAll()`: 인벤토리(슬롯+두손) 전체를 꺼내 반환하고 비움 (탈출 시 일괄 확보용)

### 2. Wagon (마차 본체)
- 적재 목록 보관(Cargo 자식 트랜스폼), **적재 수·총 가치**(보물만 합산) 계산
- `Deposit()`: 아이템/보물을 실어 확보(모델 숨김·물리 off로 보관)
- `Leave()`: 던전 종료 처리, 좌측·중앙에 결과 표시(임시 `Time.timeScale=0`)

### 3. WagonCargo / WagonLever (상호작용 지점)
- **WagonCargo**: E → 손에 든 것을 마차에 싣기(`TakeSelected` → `Deposit`)
- **WagonLever**: E → 떠나기(`Wagon.Leave()`) → 던전 종료

### 4. 마차 오브젝트 배치
- 시작 방 근처에 `Wagon`(본체 Body=WagonCargo, Lever=WagonLever) 배치

## 결과
- 보물을 마차에 실으면 확보(적재 수·가치 갱신), 떠나기 시 종료 화면(확보 보물 개수·총 가치) 표시 확인

## 추가 작업 — 살아서 탈출 시 인벤토리 보물도 확보
- **문제**: 떠나기 시 마차에 미리 실은 것만 계산되고, 손에 들고 있던(인벤토리) 보물은 누락
- **수정**: `Wagon.Leave(InventorySystem playerInventory)`로 시그니처 변경 → 떠나기 시 **플레이어 인벤토리 전체(`TakeAll()`)를 마차로 확보**한 뒤 종료 처리
- `WagonLever`가 `PlayerInteractor.Inventory`를 넘겨주도록 연결
- → 살아서 탈출하면 마차에 미리 실은 것 + 손에 들고 있던 것 **전부 포함**해 총 가치 계산

## 결과 (추가 확인)
