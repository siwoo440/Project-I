\# 35일차 개발 일지



\## 작업 주제



게임 설정 화면과 설정값 저장 구현



\## 작업 목표



\- 메인 메뉴 설정 화면 추가

\- 마우스 감도 설정

\- 전체 음량 설정

\- 전체 화면과 창 모드 설정

\- 해상도 선택

\- 수직 동기화 설정

\- 게임 재실행 후 설정 유지



\## 구현 내용



\### GameSettingsManager



게임의 공통 사용자 설정을 관리하는 싱글톤 관리자를 구현했다.



관리하는 설정은 다음과 같다.



\- 마우스 감도 배율

\- 전체 음량

\- 전체 화면 여부

\- 해상도 너비와 높이

\- 수직 동기화 여부



설정 관리자는 DontDestroyOnLoad를 사용하여 MainMenu, Village, Dungeon 사이에서 유지된다.



\### PlayerPrefs 저장



사용자 설정은 캠페인 JSON 저장과 분리하여 PlayerPrefs에 저장했다.



새 게임을 시작하거나 캠페인 저장 기록을 삭제해도 화면과 입력 및 음량 설정은 유지된다.



\### GameSettingsUI



기존 프로젝트의 임시 UI 방식에 맞춰 OnGUI 기반 설정 화면을 구현했다.



설정 화면에서 다음 기능을 사용할 수 있다.



\- 마우스 감도 슬라이더

\- 전체 음량 슬라이더

\- 해상도 이전 및 다음 선택

\- 전체 화면 토글

\- 수직 동기화 토글

\- 적용 및 저장

\- 기본값

\- 취소



\### 마우스 감도 연결



PlayerController의 기본 마우스 감도에 사용자 설정 배율을 곱하도록 수정했다.



PlayerController Inspector의 기본 감도는 그대로 유지하고 사용자 설정만 별도로 저장한다.



\### 화면 설정



현재 모니터에서 지원하는 해상도 목록을 가져오고 너비와 높이가 같은 중복 항목을 제거했다.



전체 화면은 FullScreenWindow 방식으로 적용하며, 전체 화면을 해제하면 Windowed 방식으로 적용한다.



\### 음량 설정



AudioListener.volume을 사용하여 현재 프로젝트의 모든 AudioSource에 동일한 전체 음량을 적용했다.



배경음과 효과음의 개별 조절은 이후 AudioMixer 도입 시 확장할 예정이다.
