# 18일차 : 함정 3종 및 자동 스폰

## 목표

- 플레이어에게 피해를 주는 함정 3종을 구현한다.
- 던전 생성 완료 후 각 방에 함정을 자동 배치한다.
- 방 밝기에 따라 함정 생성량을 조절한다.

## 한 일

- 가시 함정의 20 피해를 구현했다.
- 도끼 함정의 60 피해와 발동 지연을 구현했다.
- 방패를 무시하는 즉사 함정을 구현했다.
- 즉사 피해에서도 부활의 돌과 기존 사망 처리가 작동하도록 연결했다.
- 함정 발동 후 재사용 대기시간을 적용했다.
- 일회용 함정 설정을 추가했다.
- TrapSpawnManager를 구현했다.
- DungeonSpawnUtility를 이용해 안전한 함정 위치를 검색했다.
- 시작 방의 함정 생성을 제외했다.
- 어두운 방일수록 함정 생성량이 증가하도록 적용했다.
- 기존 통합 SpawnManager 스크립트를 제거했다.

## 결과

- 가시, 도끼, 즉사 함정이 플레이어에게 각각 설정된 효과를 적용한다.
- 일반 함정 피해에는 방패 감소가 적용된다.
- 즉사 함정은 방패를 무시하지만 부활의 돌은 사용할 수 있다.
- 던전 생성 후 함정이 자동으로 배치된다.
- 몬스터, 보물, 함정 생성 시스템이 각각 독립된 매니저로 관리된다.
