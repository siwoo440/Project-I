# 42일차 개발 일지

## 개발 목표

- 절차적으로 생성된 방에 함정과 상호작용 요소 자동 배치
- 방별 생성 조건과 프리팹 연결 상태 점검
- 랜덤 시드에 따른 플레이어 시작 위치 문제 해결
- 첫 번째 방을 안전한 시작 방으로 고정

## 구현 내용

### 1. 방 콘텐츠 자동 생성 시스템 점검

절차적으로 생성된 일반 방을 대상으로 함정, 보물, 미믹 등의 콘텐츠가 자동으로 배치되도록 생성 조건을 점검했다.

각 생성 관리자가 대상 방을 검색한 뒤 확률, 최소 생성 개수, 프리팹 설정에 따라 콘텐츠를 배치하도록 구성했다.

점검한 주요 항목은 다음과 같다.

- 자동 생성 허용 여부
- 대상 방 검색 결과
- 생성 확률 판정 결과
- 최소 생성 개수
- 생성 프리팹 배열
- 안전한 생성 위치
- 바닥 Layer
- NavMesh 생성 상태

### 2. 생성 위치 탐색 문제 점검

콘텐츠 생성 위치를 찾지 못하는 문제를 확인했다.

일반 방의 Ceiling에 설정된 BoxCollider가 바닥을 탐색하는 Raycast를 가로막을 가능성이 있어, Ceiling Collider를 임시로 비활성화하여 생성 여부를 테스트했다.

장기적으로는 바닥을 전용 Layer로 분리하고 생성 위치 탐색 시 해당 Layer만 검사하도록 구성하는 방향을 정했다.

### 3. 콘텐츠 생성 실패 원인 정리

콘솔 메시지를 기준으로 다음 문제를 구분할 수 있도록 점검했다.

- 대상 방이 검색되지 않는 문제
- 안전한 생성 위치를 찾지 못하는 문제
- Ceiling Collider가 Raycast를 가로막는 문제
- 바닥 Layer가 잘못 설정된 문제
- NavMesh가 생성되지 않은 문제
- 자동 생성 옵션이 비활성화된 문제
- 생성 확률 판정에서 제외된 문제
- Manager의 Prefab 배열이 비어 있는 문제

### 4. 시작 방 전용 프리팹 추가

`DungeonGenerator`에 시작 방 전용 프리팹 필드를 추가했다.

첫 번째 격자 좌표인 `(0, 0, 0)`에는 일반 방을 무작위로 선택하지 않고 `Room_Open`을 반드시 생성하도록 수정했다.

나머지 방은 기존과 동일하게 랜덤 시드와 연결 구조에 따라 일반 방 또는 계단 방을 선택한다.

### 5. 시작 방 수직 연결 방지

시작 방이 계단 방 또는 수직 연결 지점으로 선택되지 않도록 수정했다.

다음 단계에서 시작 좌표를 제외하도록 처리했다.

- 무작위 수직 이동 방향 선택
- 위층과 지하층 계단 후보 검색
- 수직 연결 생성
- 계단 방 프리팹 선택

이를 통해 시작 방은 항상 안전한 일반 방 구조를 유지한다.

### 6. 플레이어 시작 위치 개선

`Room_Open` 프리팹에 `PlayerSpawnPoint`를 추가했다.

던전 생성이 완료되면 플레이어를 `PlayerSpawnPoint` 위치로 이동시키도록 수정했다.

플레이어 이동 과정에서는 다음 처리를 적용했다.

- CharacterController 일시 비활성화
- Rigidbody 이동 속도 초기화
- Rigidbody 회전 속도 초기화
- 시작 위치와 회전값 적용
- CharacterController 재활성화

F9로 던전을 다시 생성하더라도 플레이어가 이전 위치에 남거나 벽과 오브젝트에 끼이지 않도록 개선했다.

### 7. 기존 던전 생성 기능 유지

시작 방 고정 기능을 추가하면서 기존 기능은 유지했다.

- 랜덤 시드 생성
- 경로별 시드 오프셋
- 다층 던전 생성
- 지하층 생성 보장
- 위층 생성 보장
- 계단 방 선택
- 방 연결부 개방
- F9 던전 재생성
- 생성 시작 및 완료 이벤트
- 생성된 방 목록 관리

## Unity Inspector 설정

`DungeonGenerator`의 `Starting Room Prefab` 필드에 `Room_Open`을 연결했다.

`Player` 필드에는 현재 Scene의 플레이어 루트 Transform을 연결했다.

기존 일반 방과 계단 방 프리팹 배열은 그대로 유지했다.

## 테스트 결과

- 랜덤 시드를 변경해도 시작 방이 항상 `Room_Open`으로 생성됨
- 시작 방 이름이 `Room_Open_F0_0_0` 형태로 생성됨
- 시작 방이 계단 방으로 변경되지 않음
- 시작 방에서 수직 연결이 생성되지 않음
- 플레이어가 `PlayerSpawnPoint`에서 시작함
- F9 재생성 후에도 플레이어 위치와 속도가 초기화됨
- 나머지 방은 랜덤 시드에 따라 변경됨
- 방별 콘텐츠 생성 조건과 프리팹 연결 상태를 점검함
