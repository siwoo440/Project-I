## 오늘의 목표
좌클릭으로 몬스터를 공격해 처치할 수 있게 한다. 무기별 차이는 다음 일차로 미루고, 오늘은 "때리면 죽는다"만 완성.

## 한 일

### 1. MonsterData / MonsterAI — 방어력 추가
- `MonsterData`에 `defense` 필드 추가, `MonsterAI`에 fallback 방어력 + `Defense` 프로퍼티 노출
- 데미지 공식에서 참조할 수 있도록 준비

### 2. PlayerCombat (신규)
- 좌클릭 입력(커서 잠금 상태에서만 동작) → 쿨다운(0.5초) 체크 후 공격
- 카메라 전방으로 **SphereCast**(반경 0.4) — 단순 Raycast보다 근접 명중 판정이 관대함
- **데미지 공식**: `max(1, 공격력 × 치명타배수 − 대상 방어력)` (기획서 PART 4.5.4)
- 임시 수치: 공격력 30 / 사거리 2.5 / 치명타 확률 20% / 치명타 배수 1.8배
- `MonsterAI.TakeDamage()` 호출로 11일차에 준비해둔 처치 로직과 연결

## 결과
- 좌클릭으로 몬스터 공격 시 Console에 데미지 로그(치명타 시 별도 표시) 확인
- 반복 공격으로 몬스터 체력 소진 시 처치(Destroy) 확인
- 허공/벽 공격 시 "빗나감" 로그 확인
