# 프로젝트 I — 7일차 개발 일지

> **단계**: Phase 1 — 프로토타입 (싱글 핵심 루프)
> **목표**: 던전 절차적 생성 — 방 프리팹 랜덤 조립 (기획서 PART 6.1)

---

## 오늘의 목표
매 플레이마다 다른 배치의 던전을 생성. 방 프리팹을 격자 위에서 랜덤 워크로 이어붙이고, 각 방에 6일차의 밝기 시스템(LightRoom)을 내장해 자동 결합.

## 한 일

### 1. Room (방 스크립트)
- 4방향(N/E/S/W) 벽을 가진 방 컴포넌트
- `OpenSide(Dir)` — 인접 방과 연결되는 쪽 벽을 비활성화해 통로 생성
- `Opposite(Dir)` — 연결된 두 방이 마주보는 벽을 함께 열기 위한 반대 방향 계산

### 2. DungeonGenerator (스텁 → 완성)
- **그리드 랜덤 워크** 알고리즘: 현재 칸에서 임의 방향으로 이동하며 방 생성/연결
- 시드(랜덤/고정) 지원, 방 개수·칸 크기 Inspector 설정
- 생성 후 플레이어를 시작 방(원점)으로 이동
- 재생성 시 기존 방 정리(Destroy) 후 재생성

### 3. 방 프리팹 조립 (씬 → 프리팹)
- 바닥(Floor) + 벽 4개(Wall_N/E/S/W, 12×3×0.5) + **Box Collider(Trigger, 12×4×12)**
- **LightRoom + Room** 컴포넌트 부착, Room의 벽 슬롯에 벽 4개 연결
- `Assets/Prefabs`에 프리팹화 → 생성기가 이를 인스턴스화

### 4. 기존 테스트 맵 정리
- 손으로 만든 Plane/테스트 Cube/Room1·Room2/수동 Brazier 삭제
- 이제 던전 전체가 절차적으로 생성됨

## 결과
- Play할 때마다 방 8개가 랜덤 배치·연결, 연결된 방 사이 벽이 열려 통로 형성
- 밝기 시스템(6일차)이 생성된 각 방에서 정상 작동(랜턴 없인 어둡고, 방마다 밝기 UI 갱신)
