# 24일차 : 출혈·둔화 및 회복 아이템

---

## 목표

- 플레이어에게 지속되는 상태 이상 기반을 구현한다.
- 가시 함정과 도끼 함정에 서로 다른 상태 이상을 연결한다.
- 상태 이상의 남은 시간과 효과를 임시 화면에 표시한다.
- 회복 아이템으로 체력과 상태 이상을 회복하도록 한다.

---

## 한 일

- PlayerStatusEffectSystem을 구현했다.
- 출혈의 지속시간, 피해량, 피해 간격을 관리하도록 했다.
- 출혈 피해가 방패를 무시하도록 상태 피해 API를 추가했다.
- 둔화 상태에서 플레이어 이동속도에 배율을 적용하도록 했다.
- 출혈과 둔화의 남은 시간을 OnGUI에 표시했다.
- 같은 상태가 재적용되면 더 강한 효과와 긴 지속시간을 유지하도록 했다.
- 가시 함정 발동 시 출혈을 적용하도록 했다.
- 도끼 함정 발동 시 둔화를 적용하도록 했다.
- 즉사 함정은 추가 상태 이상을 적용하지 않도록 했다.
- PlayerController에 상태 피해와 체력 회복 API를 추가했다.
- 부활하거나 최종 사망하면 모든 상태 이상을 제거하도록 했다.
- 선택한 회복 아이템을 확인하고 소모하는 인벤토리 API를 추가했다.
- PlayerInteractor에 R키 회복 아이템 사용 기능을 추가했다.
- 출혈을 제거하고 체력을 회복하는 붕대를 제작했다.
- 체력을 회복하고 출혈과 둔화를 제거하는 회복약을 제작했다.

---

## 결과

- 가시 함정을 밟으면 기본 피해와 출혈이 적용된다.
- 출혈 상태에서는 일정한 간격으로 방패를 무시하는 피해를 받는다.
- 도끼 함정을 밟으면 기본 피해와 둔화가 적용된다.
- 둔화 상태에서는 걷기와 달리기 속도가 감소한다.
- 출혈과 둔화의 남은 시간을 화면에서 확인할 수 있다.
- 붕대를 사용하면 체력이 회복되고 출혈이 제거된다.
- 회복약을 사용하면 체력이 회복되고 출혈과 둔화가 제거된다.
- 회복할 내용이 없으면 회복 아이템이 소모되지 않는다.
- 부활과 최종 사망 시 모든 상태 이상이 제거된다.

---

## 현재 제한 사항

- 상태 이상 표시는 임시 OnGUI 방식이다.
- 붕대와 회복약은 아직 자동 생성 목록에 포함되지 않는다.
- 상태 이상 전용 화면 효과와 사운드는 구현하지 않았다.
- 출혈과 둔화의 저항 수치나 면역 규칙은 구현하지 않았다.
- 회복 아이템 사용 애니메이션과 사용시간은 구현하지 않았다.
- 상태 이상과 회복 아이템의 네트워크 동기화는 구현하지 않았다.
