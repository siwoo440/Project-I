\# 20일차 : 미믹 위장 및 기습



\## 목표



\- 보물상자로 위장한 미믹을 구현한다.

\- 접근, 상호작용, 선제공격으로 위장을 해제하도록 한다.

\- 위장 해제 후 기존 MonsterAI의 추격과 공격을 사용한다.

\- 일반 보물 생성 기회에 낮은 확률로 미믹이 등장하도록 한다.



\## 한 일



\- MonsterAI에 피격 이벤트를 추가했다.

\- 외부 기믹에서 MonsterAI를 추격 상태로 전환하는 Alert 기능을 추가했다.

\- 보물상자 외형과 몬스터 외형을 전환하는 Mimic을 구현했다.

\- 플레이어 접근 거리 기반 위장 해제를 구현했다.

\- E키 상호작용 기반 위장 해제를 구현했다.

\- 위장 상태에서 공격받았을 때 즉시 모습을 드러내도록 했다.

\- 위장 해제 후 범위 안의 플레이어에게 기습 피해를 적용했다.

\- MimicData와 Monster\_Mimic 프리팹을 제작했다.

\- TreasureSpawnManager가 일정 확률로 보물 대신 미믹을 생성하도록 확장했다.

\- 던전 재생성 시 기존 미믹을 제거하도록 처리했다.

\- 사용되지 않는 기존 통합 SpawnManager를 제거했다.



\## 결과



\- 미믹이 일반 보물상자처럼 위장한다.

\- 접근, 상호작용, 피격 중 하나가 발생하면 위장을 해제한다.

\- 가까운 플레이어에게 기습 피해를 주고 추격을 시작한다.

\- 기존 무기 전투와 MonsterAI FSM을 그대로 재사용한다.

\- 보물 생성 기회마다 15% 확률로 미믹이 등장한다.
